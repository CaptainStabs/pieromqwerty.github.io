  1. Function Basics:
  1. Function Notation:
  1. Function notation is the way a function is written. It is meant to be a precise way of giving information about the function without a rather lengthy written explanation.
  2. The most popular function notation is f (x) which is read "f of x".
  1. This is NOT the multiplication of f times x..
  1. Traditionally, functions are referred to by single letter names, such as f, g, h and so on.
  2. Any letter(s), however, may be used to name a function. Examples:
  1. The f (x) notation is another way of representing the y-value in a function, y = f (x).
  2. The y-axis may even be labeled as the f (x) axis, when graphing.
  3. Ordered pairs may be written as (x, f (x)), instead of (x, y).
  1. Composition of Functions: 
  1. In mathematics, function composition is an operation that takes two functions f and g and produces a function h such that h(x) = g(f(x)). ... The resulting composite function is denoted g ∘ f : X → Z, defined by (g ∘ f )(x) = g(f(x)) for all x in X.
  1. Inverse Functions:
  1. When creating an inverse function, the x and the y (or f(x) are swapped and then the equation is put into function notation. For example, the function f(x) = 2x+3, has an inverse of (y-3)/2.

  1. Quadratic Forms:
  1. Different forms:
  1. Standard form: y = ax2 + bx + c where the a,b, and c are just numbers.
  2. Factored form: y = (ax + c)(bx + d) again the a,b,c, and d are just numbers.
  3. Vertex form: y = a(x + b)2 + c again the a, b, and c are just numbers.
  1. Converting:
  1. If we have a quadratic ax^2 + bx + x = 0 (Standard Form), with solutions x1 and x2, then we can factor the quadratic as a(x-x1)(x-x2) (Factored Form)
  2. Back to our standard form, if we factor out ax^2 and bx only to get an even factor, and then adding c to the end will get Vertex form
  3. To get from Factored or Vertex form to Standard form, just multiply it out.
